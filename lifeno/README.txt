生命靈數進階分析 App v2

本版調整：
1. 圈數強弱分析只顯示有意義的資料：
   - 缺數
   - 加強能量
   - 過強提醒
   - 若分布平均，會顯示簡短說明，不再全部列出 1～9。

2. 連線分析只顯示：
   - 已形成連線
   - 最值得補強的線：只列出已具備 2 個數字、只差 1 個數字的線。
   - 不再把所有沒有形成的線全部列出，避免混亂。

手機使用方式：
1. 上傳整個資料夾到 GitHub Pages、Netlify、Vercel 或網站空間。
2. 用手機 Chrome 或 Safari 開啟網址。
3. Android Chrome：右上角「⋮」→「加入主畫面」。
4. iPhone Safari：分享 →「加入主畫面」。

如果直接在手機本機開 index.html，可以使用分析功能，但 PWA 安裝與離線功能可能不完整。
