DISC人格分析系統 v2｜96題專業版

目錄：
student_version/
  index.html 作答頁
  report.html 學生報告頁

teacher_version/
  index.html 作答頁
  report.html 教師報告頁

personal_version/
  index.html 作答頁
  report.html 個人報告頁

功能：
- 96題，D/I/S/C各24題
- 每頁8題
- 自動暫存
- 產生報告後跳到獨立 report.html
- DISC雷達圖
- 色塊圖
- 單核心/雙核心判定
- 學生版、教師版、個人版
- 個人版包含：順境人格、壓力人格、領導風格、學習風格、人際風格、親職風格、適合職業、團隊角色
- 列印PDF最佳化

使用方式：
1. 學生版開 student_version/index.html
2. 教師版開 teacher_version/index.html
3. 個人版開 personal_version/index.html
4. 完成題目後按「產生報告」
5. 到 report.html 列印或存成PDF

上傳 GitHub Pages 時整個 DISC_v2_96 資料夾上傳即可，不需要改程式碼。


新增代表動物圖片：D獅子、I孔雀、S無尾熊、C貓頭鷹。報告頁會顯示主要代表動物；雙核心型會加上次要動物說明。
