# DISCV4 Prompt Library

Prompt Library 是 DISCV4 的內容引擎草稿區。程式負責計分與組裝報告；提示詞負責日後產生「故事、鼓勵信、教師建議、生涯分析」等 AI 內容。

目前版本先建立資料夾與提示詞範本，不直接呼叫外部 AI。未來若接 API，只要讓 AI Engine 讀取這些 prompt，即可產生更個人化的文字。

## 設計原則

1. 不貼標籤：避免「你就是……」，改用「你比較傾向……」。
2. 不說缺點：改用「可能遇到的挑戰」。
3. 文字要可行動：每段建議都要讓使用者知道可以做什麼。
4. 適合學生閱讀：學生版語氣溫暖、清楚、不說教。
5. 教師版務實：提供可在課堂使用的觀察與策略。
6. 個人版導航：聚焦工作、關係、生涯與未來行動。
