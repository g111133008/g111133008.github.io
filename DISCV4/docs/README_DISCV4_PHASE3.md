# DISCV4 Phase 3

本階段完成教師版專區，並維持「共同分析只說一次、教師版不重複學生內容」的設計原則。

## 完成項目

- 新增 `js/reports/teacher.js`
- `report.html` 載入教師版模組
- `reports/config.js` 改為呼叫 `teacherReport(data, analysis)`
- 教師版包含 9 個實用章節：
  1. 一眼看懂這位學生
  2. 常見課堂情境判讀
  3. 最佳教學策略
  4. 提問與點名方式
  5. 合作分組與角色安排
  6. 激勵與回饋方式
  7. 壓力警訊與早期介入
  8. 作業與評量調整
  9. 家長溝通提醒
- 48 題時仍會自動隱藏細項向度預留區。

## 使用方式

- 學生版：`index.html?type=student&form=quick`
- 教師版：`index.html?type=teacher&form=quick`
- 個人版：目前保留下一階段擴充。

## 設計原則

一件事情只說一次，而且說在最適合的人面前。
教師版不再重複學生版的「我的優勢」「我的學習風格」等內容，而是轉換為老師可直接使用的教學觀察與介入策略。
